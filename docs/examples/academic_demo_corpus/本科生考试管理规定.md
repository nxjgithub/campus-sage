# 本科生考试管理规定

## 一、缓考申请流程

学生因突发疾病、家庭重大事项或学校批准的公务活动无法参加期末考试时，可以申请缓考。

办理步骤如下：

1. 学生本人在考试前向所在学院提交缓考申请表。
2. 同时提交医院诊断证明、相关请假审批单或活动通知等佐证材料。
3. 学院教学秘书完成初审后，报教务处复核备案。
4. 教务处审核通过后，学生缓考资格生效，并在成绩系统中标记为“缓考”。

未在考试前完成申请且无正当证明的，不得补办缓考。

## 二、补考资格

补考通常适用于以下学生情形：

- 正常参加课程考核但成绩不及格的学生；
- 因已批准缓考而未参加原定考试、后续需要参加补考安排的学生；
- 因考试违规被取消成绩、课程明确规定不得补考的，不具有补考资格。

补考名单由学院和教务处联合核对后发布，学生应按通知时间参加考试。
